## Usage Notes

**Cold Start Strategy**: If you're new to RSS, start with 10-15 sources from categories that match your interests, then expand gradually. Don't try to follow all 92 at once.

**Quality Indicators**: These sources were selected based on HN popularity, which correlates with:
- Long-form, thoughtful content
- Technical depth
- Original research/insights
- Low commercial incentive to manipulate

**OPML Import**: The complete list is available as an OPML file at `../assets/karpathy-hn-2025.opml` for direct import into RSS readers like NetNewsWire, Feedly, or Inoreader.
