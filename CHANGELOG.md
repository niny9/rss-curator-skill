# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-07

### Added
- Initial release of RSS Curator Skill
- Karpathy's 92 HN popular sources (2025) integrated
- 6-dimension quality assessment framework
- OPML import/export functionality
- Source recommendation system
- Complete documentation in English and Chinese
- 4 reference documents:
  - `karpathy-sources.md` - Categorized source catalog
  - `source-quality.md` - Quality evaluation framework
  - `opml-format.md` - OPML specification and guides
  - `ai-processing.md` - AI processing workflows
- Test cases and test report
- MIT License
- Contributing guidelines

### Features
- **Source Management**: Import 92 curated sources instantly
- **Quality Evaluation**: 6-dimension scoring system (Originality, Depth, Frequency, Independence, Expertise, Longevity)
- **Smart Recommendations**: Find similar sources based on characteristics
- **OPML Support**: Generate and import OPML files for all major RSS readers
- **Progressive Disclosure**: Load detailed references only when needed
- **Bilingual Support**: Full documentation in English and Chinese

### Documentation
- Comprehensive README with quick start guide
- Detailed user guides for import, quality assessment, and recommendations
- Test cases covering all major workflows
- Complete test report with 10/10 scores

### Quality
- All 92 sources verified and categorized
- Quality scoring examples (Simon Willison: 27/30, Paul Graham: 30/30)
- Starter pack with 15 hand-picked sources for beginners
